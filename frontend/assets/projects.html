Данный проект представляет собой клиент-серверный кликер. Целью игры является нажать наибольшее количество раз за 10 секунд. Подсчет и таймер осуществляются за счет самописных функций. По окончанию игры, результат отправляется на сервер через ajax запрос. Происходит обновление профиля пользователя, таблицы результатов и перенаправление на страницу результатов, формирующуюся посредством сложных запросов к БД. Также в приложении присутствует система регистрации, авторизации, смены и восстановления пароля и таблица лидеров.

Сайт с картинками шайб

Главная страница

Лайк изображения

Previous Next

Данный сайт представляет собой одностраничник для поиска отображения случайных картинок. При каждом гет запросе сайт обращается к публичному API и получает оттуда картинку. При нажатии кнопки скачать - картинка, полученная от API, сохраняется на сервер, после чего формируется HttpResponse, содержащий данную картинку. Как только картинка была передана пользователю - она удаляется с сервера. На сайте также продемонстрирована работа с сессиями (счетчик просмотров) и присутствует система авторизации. Также авторизированный пользователь может лайкнуть изображение, чтобы сохранить его в избранное.
